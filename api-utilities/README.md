# API Utilities — Быстрые утилиты Ozon и WB

Три однофайловых скрипта для быстрой проверки данных через API маркетплейсов.  
Исходники: `C:\Users\Admin\`

> **Внимание:** В оригинальных файлах API-ключи вшиты напрямую в код.  
> В этом репозитории они заменены на `os.environ.get(...)` — перед запуском  
> установи переменные окружения или пропиши ключи локально.

---

## Скрипты

### `check_ozon_stocks.py` — остатки Ozon

Быстро проверяет остатки товаров на складах Ozon.

**Что делает:**
1. Запрашивает список первых 10 товаров через `POST /v2/product/list`
2. Берёт `offer_id` первых 5 товаров
3. Проверяет остатки через `POST /v4/product/info/stocks`
4. Выводит результат в консоль

**Требует:**
```bash
set OZON_CLIENT_ID=ваш_client_id
set OZON_API_KEY=ваш_api_key
```

**Запуск:**
```bash
cd C:\Users\Admin
uv run check_ozon_stocks.py
```

---

### `fetch_wb_commissions.py` — комиссии Wildberries

Получает актуальные комиссии WB по всем категориям.

**Что делает:** Запрашивает через WB API таблицу комиссий — категория → % комиссии. Выводит в консоль.

**Зачем:** Перед расчётом цены нужно знать точную комиссию для категории товара.

**Запуск:**
```bash
uv run fetch_wb_commissions.py
```

---

### `fetch_wb_tariffs.py` — тарифы Wildberries

Получает тарифы WB на логистику: хранение, приёмка, доставка.

**Что делает:** Запрашивает актуальные тарифы через WB API. Нужно для расчёта unit-экономики — сколько стоит хранить и доставлять единицу товара.

**Запуск:**
```bash
uv run fetch_wb_tariffs.py
```

---

## Когда использовать эти утилиты

| Задача | Скрипт |
|--------|--------|
| Быстро проверить остатки на Ozon | `check_ozon_stocks.py` |
| Уточнить % комиссии WB для новой категории | `fetch_wb_commissions.py` |
| Посчитать стоимость логистики WB | `fetch_wb_tariffs.py` |

Для постоянной работы с данными используй полные системы:
- `autoparts-ecommerce/` или `ifoam-ecommerce/` — там всё хранится и ведётся история.
