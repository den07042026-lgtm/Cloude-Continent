# Image Scrapers — Скачивание фото от поставщиков

Скрипты для скачивания фотографий автозапчастей с B2B-сайтов поставщиков.  
Исходники: `C:\Users\Admin\Desktop\`  
Запуск: `uv run <script>.py`

---

## Скрапер-скрипты

| Скрипт | Источник | Описание |
|--------|---------|---------|
| `avtoliga_images.py` | b2b.avtoliga.ru | Скачивает фото для позиций с поставщиком Автолига из Топ-500 ВБ |
| `autopiter_images.py` | autopiter.ru | Скачивает фото с Autopiter |
| `stparts_images.py` | stparts.ru | Скачивает фото с Stparts |
| `mikado_images.py` | Локальный файл | Копирует фото из локальных данных Mikado |
| `mikado_web_images.py` | mikado-parts.ru | Скачивает фото напрямую с сайта Mikado |
| `avtoliga_via_mikado.py` | Авто + Mikado | Комбо: сначала ищет в Mikado, при отсутствии — в Авто |

Все скрипты работают с файлом `Топ-500 ВБ_new.xlsx` на Desktop.

---

## `avtoliga_images.py` — подробно

1. Читает `Топ-500 ВБ_new.xlsx` — строки где `Поставщик = Автолига`
2. Логинится на `b2b.avtoliga.ru` через Selenium
3. Ищет каждую позицию по артикулу
4. Скачивает все доступные фото в `Топ-500 ВБ\Изображения Автолига\{артикул}\`
5. Красит ячейки в Excel: зелёный = скачано, красный = не найдено

**Настройки в начале файла:**
```python
EXCEL_PATH  = r"C:\Users\Admin\Desktop\Топ-500 ВБ\Топ-500 ВБ_new.xlsx"
IMAGES_DIR  = Path(r"C:\Users\Admin\Desktop\Топ-500 ВБ\Изображения Автолига")
LOGIN_EMAIL = "Control.vlz@gmail.com"
```

## `ЗАПУСТИТЬ_Автолига.bat` — интерактивное меню

Запускает меню выбора источника:
```
1 → avtoliga_images.py      (b2b.avtoliga.ru)
2 → mikado_images.py        (локальный файл)
3 → mikado_web_images.py    (mikado-parts.ru)
```

## Утилиты для работы с данными

| Скрипт | Описание |
|--------|---------|
| `check_empty.py` | Находит ячейки без данных в Excel-файле |
| `check_price.py` | Проверяет корректность цен (формат, диапазон) |
| `check_structure.py` | Проверяет структуру Excel: порядок столбцов, типы данных |
| `check2.py` | Дополнительные проверки целостности данных |
| `cleanup_avtoliga.py` | Очищает данные от Автолига: лишние символы, форматирование |
| `debug_mikado.py` | Отладочный вывод данных Mikado для конкретного артикула |
| `fix_excel_wrap.py` | Исправляет перенос текста в ячейках (убирает лишние переносы) |
| `recolor_excel.py` | Перекрашивает ячейки по условию (статус, наличие фото и т.д.) |
| `trim_articles.py` | Обрезает пробелы и лишние символы в артикулах |
| `wb_autoparts_categories.py` | Выводит список категорий WB для автозапчастей |
