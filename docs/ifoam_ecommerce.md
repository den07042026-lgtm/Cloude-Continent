# IFOAM Ecommerce Analytics System

**Путь:** `C:\Users\Admin\Documents\Ecommerce\`  
**Язык:** Python 3.10+  
**Статус:** Активный

---

## Что это

Аналитическая система для управления продажами компании **IFOAM** на маркетплейсах **Ozon** и **Wildberries**.

По архитектуре идентична Autoparts_Ecommerce (тот же 5-слойный пайплайн), но адаптирована под конкретные SKU и категории IFOAM.

Дополнительно содержит папку `ozon_prompts_v2/` — полный набор скриптов для генерации контента карточек товаров: ТЗ, SEO, описания, ценообразование, слайды.

---

## Архитектура (5 слоёв)

Идентична Autoparts_Ecommerce:
```
modules/
├── 01_collect/     → Ozon FBO/FBS, начисления, остатки; WB финансы, реклама
├── 02_process/     → Нормализация, объединение Ozon + WB
├── 03_analyze/     → Прибыль, маржа, анализ остатков
├── 04_forecast/    → Прогноз спроса на 6 мес., нормы запасов
└── 05_report/      → Ежедневный и управленческий отчёты
```

## Папка `ozon_prompts_v2/` — генерация контента

Отдельная система скриптов для создания полного контента карточек Ozon.

### SEO и исследование
| Скрипт | Что делает |
|--------|-----------|
| `05_seo_fetcher.py` | Собирает SEO-данные из Ozon (частотность запросов, конкуренты) |
| `analyze_wb.py` | Анализирует данные WB для SEO |
| `analyze_history.py` | Анализирует историю продаж |

### Генерация ТЗ
| Скрипт | Что делает |
|--------|-----------|
| `gen_tz.py` | Генерирует ТЗ для фотографа/дизайнера |
| `gen_tz_product.py` | ТЗ на продукт |
| `gen_tz_photographer.py` | ТЗ для фотографа |
| `gen_tz_pptx_119105.py` | ТЗ в формате PowerPoint для артикула 119105 |

### Слайды (инфографика)
| Скрипт | Что делает |
|--------|-----------|
| `add_photo_slides.py` | Добавляет слайды с фото в презентацию |
| `rebuild_photo_slides.py` | Перестраивает слайды |
| `fix_slide5.py` | Исправляет слайд 5 |
| `fix_slide13.py` | Исправляет слайд 13 |

### Ценообразование
| Скрипт | Что делает |
|--------|-----------|
| `make_pricing.py` | Расчёт цен v1 |
| `make_pricing_v2.py` | Расчёт цен v2 (актуальный) |
| `run_pricing_119105.py` | Запуск ценообразования для арт. 119105 |

### Листинг
| Скрипт | Что делает |
|--------|-----------|
| `gen_listing_119105.py` | Генерирует листинг для артикула 119105 |
| `save_helpers.py` | Вспомогательные функции сохранения |
| `add_reference_table.py` | Добавляет справочную таблицу |

## Главный файл `ozon_pricing.py`

Расчёт оптимальной цены для FBS с учётом:
- Комиссии Ozon по категории
- Стоимости логистики (по весу и габаритам)
- Себестоимости товара
- Целевой маржи

## Как запустить

```bash
cd C:\Users\Admin\Documents\Ecommerce

python orchestration/pipeline.py

# Ценообразование
python ozon_pricing.py

# Генерация SEO
python ozon_prompts_v2/05_seo_fetcher.py
```
