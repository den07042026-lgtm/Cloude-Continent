## Что это

Набор скриптов для подготовки батча товаров к загрузке на Wildberries.  
Название: **"Топ ВБ 1306"** = Топ-500 WB, подготовка от 13.06.2026.

Задача: взять данные из Mikado, обогатить их через AI (Gemini), проверить, заполнить все поля в Excel, подготовить к загрузке.

## Как работает `gen_descriptions.py`

Самый важный скрипт — генерация описаний через Gemini.

1. Читает `Топ-500 ВБ 1306.xlsx`
2. Находит строки где `col10` (описание) пустая
3. Смотрит на соседние строки с описаниями — извлекает стиль
4. Формирует промпт для Gemini в стиле существующих описаний
5. Записывает сгенерированные описания обратно в Excel

**Требует:** `GEMINI_API_KEY` из `C:\Users\Admin\Documents\Autoparts_Ecommerce\.env`

```bash
cd "C:\Users\Admin\Desktop\Топ ВБ 1306"
uv run gen_descriptions.py
```

## Типичный рабочий процесс батча

```
1. mikado_fill.py        → заполнить данные из Mikado
2. normalize_names.py    → нормализовать названия
3. normalize_alts.py     → нормализовать альты
4. check_descr.py        → найти строки без описаний
5. gen_descriptions.py   → сгенерировать описания через Gemini
6. collect_images_1406.py → собрать фото
7. highlight_empty_1406.py → проверить что всё заполнено
8. fix_headers_1406.py   → финальная проверка заголовков
9. → загрузить на WB
```
