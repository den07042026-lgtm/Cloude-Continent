# Топ ВБ 1306 — Подготовка батча WB (13–14 июня 2026)

Набор скриптов для подготовки Топ-500 позиций к загрузке на Wildberries.  
Исходники: `C:\Users\Admin\Desktop\Топ ВБ 1306\`  
Главный файл данных: `Топ-500 ВБ 1306.xlsx`

---

## Все скрипты — полный справочник

### Данные из Mikado
| Скрипт | Описание |
|--------|---------|
| `mikado_fill.py` | Заполняет поля (OEM, размеры, совместимость) данными из базы Mikado |
| `show_mikado_descr.py` | Показывает описания из Mikado для ревью и сравнения |

### Нормализация данных
| Скрипт | Описание |
|--------|---------|
| `normalize_names.py` | Нормализует названия товаров к единому формату |
| `normalize_alts.py` | Нормализует альтернативные наименования |
| `clean_base.py` | Очищает базу от мусорных символов, дублей, пустых строк |
| `preview_names.py` | Предпросмотр нормализованных названий без сохранения |

### AI-генерация описаний
| Скрипт | Описание |
|--------|---------|
| `gen_descriptions.py` | Генерирует описания через **Gemini 2.5 Flash** для строк без описания. Подбирает стиль по соседним заполненным строкам |
| `list_models.py` | Выводит список доступных Gemini-моделей |
| `test_gemini.py` | Тест подключения к Gemini API (проверка ключа) |

### Заполнение полей — батч 14.06 (`fill_*_1406.py`)
| Скрипт | Описание |
|--------|---------|
| `fill_dimensions_1406.py` | Заполняет габариты и вес |
| `fill_descriptions_1406.py` | Заполняет описания из сгенерированных данных |
| `fill_compat_1406.py` | Заполняет совместимость (применяемость по маркам/моделям авто) |
| `fill_oem_1406.py` | Заполняет OEM-номера |
| `fill_params_1406.py` | Заполняет дополнительные параметры |

### Сбор изображений
| Скрипт | Описание |
|--------|---------|
| `collect_images_1406.py` | Собирает ссылки на фото для батча 14.06 из доступных источников |
| `collect_images_fullscan.py` | Полный скан всех источников фото по всем позициям |
| `collect_images_stemmatch.py` | Подбирает фото по стемм-совпадению артикулов (нечёткий поиск) |
| `fill_images_autopiter.py` | Заполняет ссылки на фото с сайта Autopiter |
| `stparts_images_1406.py` | Скачивает фото с Stparts для батча 14.06 |

### Проверки (`check_*.py`)
| Скрипт | Описание |
|--------|---------|
| `check_headers.py` | Проверяет корректность заголовков столбцов в Excel |
| `check_descr.py` | Находит строки без описания |
| `check_images_col.py` | Проверяет колонку с изображениями (формат, наличие) |
| `check_multi_img.py` | Проверяет позиции с несколькими изображениями |
| `check_suppliers.py` | Проверяет корректность данных о поставщиках |
| `check_row1.py` | Проверяет первую строку (заголовки шаблона WB) |

### Исправления (`fix_*.py`)
| Скрипт | Описание |
|--------|---------|
| `fix_freeze.py` | Исправляет закрепление строк/столбцов в Excel |
| `fix_scroll.py` | Исправляет проблемы с прокруткой листа |
| `fix_headers_1406.py` | Исправляет заголовки столбцов для батча 14.06 |

### Подсветка и визуальный контроль
| Скрипт | Описание |
|--------|---------|
| `highlight_empty.py` | Подсвечивает незаполненные ячейки красным |
| `highlight_empty_1406.py` | То же для батча 14.06 |
| `highlight_missing_images_1406.py` | Подсвечивает позиции без фото |

### Служебные
| Скрипт | Описание |
|--------|---------|
| `copy_headers.py` | Копирует заголовки из одного листа/файла в другой |
| `inspect_xml.py` | Инспектирует XML-структуру внутри .xlsx файла (для отладки) |

## Типичный рабочий процесс батча

```
1.  mikado_fill.py              → заполнить данные из Mikado
2.  normalize_names.py          → нормализовать названия
3.  normalize_alts.py           → нормализовать альтернативы
4.  check_descr.py              → найти строки без описания
5.  gen_descriptions.py         → сгенерировать описания через Gemini
6.  fill_compat_1406.py         → заполнить совместимость
7.  fill_oem_1406.py            → заполнить OEM
8.  fill_dimensions_1406.py     → заполнить габариты
9.  collect_images_1406.py      → собрать фото
10. highlight_empty_1406.py     → проверить незаполненные поля
11. highlight_missing_images_1406.py → проверить позиции без фото
12. fix_headers_1406.py         → финальная проверка заголовков
13.                             → загрузить на WB
```
