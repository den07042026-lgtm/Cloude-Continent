  • 📥

    Скачать шаблон категории

    ozon_downloader.py → автоматически скачивает нужный Excel с seller.ozon.ru

    fill_ozon_templates.py → данные + фото с Яндекс.Диска

  • 📊

    Подготовить Excel-батч

    Синхронизация остатков

    wb_stock_sync.py → автообновление остатков по API

    FBO / FBS

    ozon_fbo.py, ozon_fbs.py — оба формата, система понимает оба
