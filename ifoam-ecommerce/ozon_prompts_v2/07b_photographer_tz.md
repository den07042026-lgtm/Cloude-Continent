## ПРАВИЛА ГРУППИРОВКИ В БЛОКИ

Сгруппируй карточки дизайнера в блоки съёмки по типу сцены.
Типичная структура (адаптируй под конкретный товар):

- Блок 1: ПРОДАКТ-ШОТ — товар на однородном фоне (для главного фото и карточек характеристик)
- Блок 2: ПРОЦЕСС ПРИМЕНЕНИЯ — пошаговые кадры использования товара (для карточки инструкции)
- Блок 3: РЕЗУЛЬТАТ / ЭФФЕКТ — визуальная демонстрация выгоды или эффекта
- Блок 4: БЕЗОПАСНОСТЬ — кадры с акцентом на правильное использование и предупреждения
- Блок 5: ПРОИЗВОДСТВО И ДОВЕРИЕ — цех, лаборатория, сертификаты (съёмка на производстве)

Если у дизайнера есть карточки «До/После», «Сравнение» или «Lifestyle» — добавь
отдельный блок под них.

## ЧТО ДОЛЖНО БЫТЬ В КАЖДОМ БЛОКЕ

Для каждого блока укажи:
1. Цель блока — что эти кадры дают дизайнеру и что должен понять покупатель
2. Фон — цвет, материал, текстура. Вывести из ТЗ дизайнера: если там тёмный фон — тёмный фон
3. Освещение — схема (студийное / естественное), есть ли цветные гели/акценты
4. Реквизит — полный список: упаковка, инструменты, перчатки, автомобиль и т.д.
5. Список кадров — для каждого кадра:
   - Короткое кодовое название (для именования файлов, латиница+подчёркивания)
   - Подробное описание: что в кадре, как расположено, что в фокусе
   - Ракурс: прямо / сверху / 3/4 / сбоку / детальный / снизу
   - Примечания: детали которые нельзя упустить (читаемость этикетки, руки в перчатках и т.д.)

## ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ (стандартные — можно уточнить под продукт)

Разрешение:     минимум 4 000 px по короткой стороне (24+ Мп)
Форматы:        RAW (архив) + TIFF 16-bit (мастер) + JPEG (превью)
Цветовой профиль: Adobe RGB для TIFF, sRGB для JPEG
Баланс белого:  ручной по серой карте, единый внутри каждого блока
ISO:            минимально возможное (рекомендуется ISO 100–400)
Постобработка:  ретушь пыли/царапин на упаковке, цвет жидкости/продукта — по бренд-цвету из ТЗ дизайнера

## ЧЕКЛИСТ ПРИЁМКИ

Сформируй чеклист из 10–12 пунктов на основе требований конкретного товара.
Обязательные пункты:
- Все блоки съёмки переданы
- Фон соответствует ТЗ дизайнера (тёмный / белый / предметный)
- Упаковка чистая, этикетка читается, крышка/горлышко целые
- Баланс белого единый внутри каждого блока
- Разрешение не менее 4 000 px по короткой стороне
- Файлы названы по схеме: [БРЕНД]_[АРТИКУЛ]_[БЛОК]_[КАДР]_v[N].[формат]
- Переданы все форматы: RAW + TIFF + JPEG
- Блок производства — только реальные кадры, без стоков и AI

## ИМЕНОВАНИЕ ФАЙЛОВ
