ДОКУМЕНТ: ТЕХНИЧЕСКОЕ ЗАДАНИЕ
НАЗВАНИЕ: Анализ и запуск карточки товара [Бренд] [Название] [Объём] на Ozon (арт. [Артикул])
ДАТА: [текущая дата]

   Заказчик: ifoamHOME — менеджер по товарному контенту
   Исполнитель: Аналитик / Claude Code + оператор
   Кто будет использовать: менеджер контента, дизайнер, фотограф, SEO-специалист

   Провести полный предстартовый анализ товара [Название] (арт. [Артикул]) и
   сформировать комплект документов для запуска конкурентоспособной карточки на Ozon.
   Итог: [N] выходных документов, готовых к передаче в работу дизайнеру, фотографу
   и оператору Ozon Seller.

3. ВВОДНЫЕ ДАННЫЕ
   Таблица (#, Название, Тип, Источник):
   — Этикетка товара (уже обработана → 01_card_[арт].docx)
   — История продаж XLSX (оператор скачивает из DataLens)
   — Топ конкурентов XLSX (seller.ozon.ru → Аналитика → Конкуренты → Экспортировать)
   — PDF отзывов конкурентов × 5 (Chrome → Ctrl+P)
   — PDF поисковых запросов (seller.ozon.ru → Все запросы → Ctrl+P)
   — PDF карточек конкурентов × 5 (Chrome → Ctrl+P)
   — combined_reference_normalized.csv (D:/Аналитика/Система/)
   — ozon_products_*.xlsx свежайший (D:/Аналитика/Система/Управленческий_отчет_IFOAM_ОЗОН/Товары/)
   — Папка продукта с этикеткой и фото конкурентов

4. ЭТАПЫ ВЫПОЛНЕНИЯ
   Таблица (№, Этап, Результат, Срок):
   Шаг 1  — уже выполнен: 01_card_[арт].docx
   Шаг 1a — настоящий документ: 01a_tz_[арт].docx
   Шаг 1б — инструкция Ozon Seller (Промпт 1b): гайд для оператора
   Шаг 2  — история продаж (Промпт 2): 02_sales_history_[арт].docx
   Шаг 3  — конкуренты + цены (Промпты 3, 3b): .xlsx + .docx
   Шаг 4  — боли покупателей (Промпт 4): 04_pains_[арт].docx
   Шаг 5  — SEO (Промпт 5): 05_seo_[арт].xlsx + .docx
   Шаг 6  — визуальный анализ (Промпт 6): 06_visual_[арт].docx
   Шаг 7  — ТЗ дизайнера + фотографа (Промпты 7, 7b): .docx + .pptx
   Шаг 8  — листинг + инструкция оператору (Промпты 8, 8b): .docx

5. ЗАДАЧА СЧИТАЕТСЯ ВЫПОЛНЕННОЙ, КОГДА:

## КАК РАБОТАЕТ СКРИПТ ГЕНЕРАЦИИ

Промпт 1a вызывает функцию `build_product_tz()` из модуля
`C:/Users/1/Downloads/ozon_prompts_v2/gen_tz_product.py`

Функция принимает данные из карточки товара и генерирует
полноструктурированный DOCX по Шаблон_ТЗ.docx.

Запуск:
  python3 C:/Users/1/Downloads/ozon_prompts_v2/gen_tz_product.py \
    --arт 770758 \
    --brand "ifoamHOME" \
    --name "КондиционерBalmy_РайскиеЦветы" \
    --volume "1 л" \
    --date "31.03.2026"
