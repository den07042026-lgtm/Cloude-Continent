## ИНСТРУКЦИЯ ДЛЯ ОПЕРАТОРА

### Что скачать
Файл с историей продаж по конкретному товару из DataLens.

### Когда нужно
Только если хочешь получить самые свежие данные.
Если данные не нужны свежее чем Combined (D:/Аналитика/Система/combined_marketplaces_with_profit.xlsx) —
этот шаг можно пропустить, Промпт 2 возьмёт данные автоматически.

### Шаги

1. Открой в браузере (залогинься если нужно):
   https://datalens.ru/wizard/psus7vgwxgdi9-unit-ekonomika-sistema-tablica-unit-dlya-claude

2. В средней колонке ("Столбцы") прокрути вниз до раздела "Фильтры"

3. Фильтр Артикул_6:
   - Кликни на "Артикул_6"
   - В диалоге: если в "Выбраны" есть старые значения — нажми "Очистить"
   - В колонке "Доступны" → поле "Найти" → введи числовой артикул (например 119105)
   - Кликни на найденное число → оно переместится в "Выбраны"
   - Нажми "Применить фильтр"

4. Фильтр Период:
   - Кликни на "Период"
   - В колонке "Выбраны" найди строку с текущим месяцем (например "2026-03-01 – 2026-03-31")
   - Нажми "Удалить" рядом с ней
   - Все остальные месяцы оставь
   - Нажми "Применить фильтр"

5. Экспорт:
   - В правом верхнем углу таблицы нажми "⋮" (три точки)
   - Выбери "XLSX"
   - Дождись скачивания

6. Передай Промпту 2:
   - Сообщи путь к скачанному файлу (обычно C:/Users/1/Downloads/[имя файла].xlsx)
   - Файл не переименовывай — Промпт 2 найдёт его автоматически
