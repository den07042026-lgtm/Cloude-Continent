# ПРОМПТ 8b — Инструкция по обновлению листинга в Ozon Seller
# Запускать в: любой Claude (desktop / web / Claude Code)
# Вход: ГОТОВЫЙ ЛИСТИНГ из Промпта 8
# Выход: ГАЙД ДЛЯ ОПЕРАТОРА — что и куда вставить вручную в Ozon Seller
# Оператор открывает seller.ozon.ru и копирует значения из инструкции

---

## ПРОМПТ

```
=== ВХОД: ГОТОВЫЙ ЛИСТИНГ ===
[ВСТАВЬ СЮДА полный вывод из Промпта 8]
=============================

На основе листинга составь пошаговую инструкцию для оператора.
Оператор откроет Ozon Seller вручную и будет копировать значения.

Путь: seller.ozon.ru → Товары → найти по артикулу → карандаш (редактировать)

=== ИНСТРУКЦИЯ ДЛЯ ОПЕРАТОРА: ОБНОВЛЕНИЕ ЛИСТИНГА ===

ШАГ 1 — Шаг "Информация о товаре"

  НАЗВАНИЕ — скопируй и вставь:
  ┌─────────────────────────────────────────────────────┐
  [вариант А из листинга]
  └─────────────────────────────────────────────────────┘
  Если не прошёл валидацию → используй вариант Б:
  ┌─────────────────────────────────────────────────────┐
  [вариант Б из листинга]
  └─────────────────────────────────────────────────────┘

  ОПИСАНИЕ — скопируй и вставь полностью:
  ┌─────────────────────────────────────────────────────┐
  [полный текст описания из листинга]
  └─────────────────────────────────────────────────────┘

  АННОТАЦИЯ (если поле есть, до 500 символов) — скопируй:
  ┌─────────────────────────────────────────────────────┐
  [первый абзац описания]
  └─────────────────────────────────────────────────────┘

→ Нажать "Далее"

ШАГ 2 — Шаг "Характеристики"
Обновить только то что изменилось по сравнению с Промптом 1b:

  [перечислить поля если что-то добавилось: ТН ВЭД, состав, срок годности и т.д.]

→ Нажать "Далее"

ШАГ 3 — Хэштеги (если поле есть на любом шаге)
Введи по одному, нажимая Enter после каждого:
  [список хэштегов из листинга, каждый на новой строке]

ШАГ 4 — Медиа
Ничего не менять → "Далее"

ШАГ 5 — Предварительный просмотр → "Сохранить"

После сохранения:
- Проверь контент-рейтинг (ожидаемый рост: описание даёт до 25 баллов)
- Если название не прошло — проверь: нет ли заглавных букв в середине слов,
  запрещённых символов, превышения 200 символов
======================================================
```
