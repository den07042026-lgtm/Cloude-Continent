#   Слайд 1:       Общие требования (форматы, палитра, запрещено)
#   Слайды 2–N+1:  Карточки 1..N (по одному слайду на каждую)

```

================================================

На основе всех данных выше составь ТЗ для дизайнера.

ОБЩИЕ ТРЕБОВАНИЯ (заполни из карточки товара):
- Формат: 1200×1200 px (обложка), 1200×1600 px (инфографика)
- Качество: 300 dpi, RGB
- Фон: [цвет из карточки/этикетки]
- Акцентный цвет: [из этикетки]
- Цвет текста: белый или контрастный к фону
- Логотип: правый верхний угол
- Минимальный размер шрифта: 24px
- Запрещено: лица людей (без согласования), стоковые фото, текст поверх товара,
             сравнения цен с конкурентами, фразы «лучший»/«№1» без подтверждения

Для каждой карточки используй формат:
━━━━━━━━━━━━━━━━━━━━━━━━
Карточка №: [название]
Цель: [что должен понять/почувствовать покупатель]
Фон: [описание фона, атмосфера]
Композиция: [подробное описание сцены, ракурс, детали]
Текстовые блоки:
  [метка]         | [описание текста/шрифта/расположения]
  [метка]         | [описание]
Справка конкурентов: [что делают конкуренты по этой карточке]
Обязательно (✓): [перечисли что делать]
Запрещено (✗):   [перечисли что НЕ делать]
━━━━━━━━━━━━━━━━━━━━━━━━

Обязательные карточки (минимум 9):
1. Главное фото (обложка) — CTR
2. Ключевые характеристики / УТП (инфографика)
3. Пошаговое применение (4 шага)
4. Экономика расхода (концентрат → выгода в цифрах)
5. Lifestyle / Образ жизни (эмоция, аудитория)
6. Состав и безопасность
7. Коллекция / линейка продуктов
8. Рекомендации и дозировка
9. Производство и доверие

Для каждой карточки также подготовь раздел «Примеры конкурентов»:
  — 4 конкурента + краткое описание их подхода к этому типу фото
  — ✓/✗ выводы для нашей карточки
```
