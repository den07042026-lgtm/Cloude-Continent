По данным из СПИСКА КОНКУРЕНТОВ раздели товары на сегменты по цене (экв. моего объёма):

Определи границы сегментов самостоятельно на основе распределения цен.
Типичные ориентиры:
  Масс-маркет   — нижняя треть цен, высокий объём продаж
  Средний       — середина рынка
  Премиум       — верхняя треть цен, выше среднего по марже

Для каждого сегмента укажи:
  - Ценовой диапазон (экв. моего объёма)
  - Представители (названия конкурентов)
  - Средняя выручка/мес в сегменте
  - Среднее количество отзывов
  - Доля сегмента в суммарной выручке топ-10 (%)
  - Типичные характеристики позиционирования

ЧАСТЬ 2 — ЦЕНОВЫЕ СЦЕНАРИИ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
На основе истории продаж и данных конкурентов рассчитай сценарии.

Данные для расчёта брать из справочника Ozon (автодополнение выше):
  Комиссия %  = sales_percent_fbo (или fbs — в зависимости от схемы)
  Логистика   = fbo_deliv_to_customer (фактическая, не 250 ₽ по умолчанию)
  Эквайринг   = acquiring (% от цены)

Для каждого сценария рассчитай:
  Выручка           = цена × прогноз продаж
  Себестоимость     = cost × шт
  Комиссия          = цена × sales_percent_fbo/100 × шт
  Логистика         = fbo_deliv_to_customer × шт
  Эквайринг         = цена × acquiring/100 × шт
  Прибыль           = Выручка − Себест − Комиссия − Логистика − Эквайринг
  Маржа %           = Прибыль / Выручка × 100

  Эффективная цена для покупателя Ozon Банк:
  = цена − субсидия_ozon_банк  (субсидия = avg_customer_price_other − avg_customer_price_ozon_bank)
  → указывать в сценарии как справочно: "покупатель Ozon Банк видит [X] ₽"

Прогноз продаж для каждого сценария — из истории:
  - При цене в сегменте масс-маркет → объём из истории при низких ценах
  - При цене в среднем сегменте → объём из истории при средних ценах
  - При цене в премиум → объём из истории при высоких ценах
  - Если истории нет → ориентируйся на объём продаж конкурентов в этом ценовом диапазоне

Сценарии (минимум 9, шаг ~100–200 ₽):
| Цена | Цена Ozon Банк | Сегмент | Прогноз шт/мес | Выручка | Комиссия % | Эквайринг | Логистика | Прибыль/шт | Маржа % | Ориентир конкурент |
