# ПРОМПТ 1b — Инструкция по заполнению характеристик в Ozon Seller
# Запускать в: любой Claude (desktop / web / Claude Code)
# Вход: КАРТОЧКА ТОВАРА из Промпта 1
# Выход: ГАЙД ДЛЯ ОПЕРАТОРА — пошаговая инструкция что вставить в каждое поле
# Оператор вручную открывает seller.ozon.ru и заполняет поля по этой инструкции

---

## ПРОМПТ

```
=== ВХОД: КАРТОЧКА ТОВАРА ===
[ВСТАВЬ СЮДА полный вывод из Промпта 1]
=============================

На основе карточки товара составь пошаговую инструкцию для заполнения
характеристик товара в Ozon Seller. Оператор откроет страницу вручную
и будет копировать значения из твоей инструкции.

Путь в Ozon Seller:
seller.ozon.ru → Товары → Список товаров → найти по артикулу → карандаш (редактировать)
→ Шаг 1 "Информация о товаре" → Шаг 2 "Характеристики"

Составь инструкцию в формате:

=== ИНСТРУКЦИЯ ДЛЯ ОПЕРАТОРА: ХАРАКТЕРИСТИКИ ===

ШАГ 1 — Информация о товаре
  Название товара: [оставить пока пустым — заполнится в Промпте 8b]
  Описание:        [оставить пока пустым — заполнится в Промпте 8b]
  → Нажать "Далее"

ШАГ 2 — Характеристики
Для каждого поля укажи точное значение для вставки.
Используй маппинг:

  Бренд                  → [значение из карточки]
  Название модели        → [значение из карточки]
  Класс опасности        → [если pH > 11: "Слабоопасное вещество", иначе "Не опасен"]
  Вес товара, г          → [объём/масса из карточки в граммах]
  Назначение             → [подходящий вариант из: Очистка / Мойка / Уход]
  Область применения     → [перечисли: Кузов / Диски и шины / Двигатель — по карточке]
  Сезон                  → [если круглогодичное — "На любой сезон"]
  Цвет товара            → [по акцентному цвету этикетки]
  Вид техники            → [Легковые автомобили + всё применимое из карточки]
  Упаковка               → [тип упаковки из карточки]
  Страна-изготовитель    → [страна производства из карточки]
  Комплектация           → [форма выпуска: например "Автошампунь"]
  Объём / масса          → [объём из карточки]

  Если в карточке есть поля pH, состав, срок годности, ТУ — укажи их тоже.

  Поля которые НЕ заполнять сейчас (ждут Промпт 8b):
  - Название товара
  - Описание / аннотация
  - Хэштеги / ключевые слова

→ Нажать "Далее" → на шаге "Медиа" ничего не менять → "Далее" → "Сохранить"

После сохранения проверить контент-рейтинг — он должен вырасти.
=================================================

Если какое-то поле из карточки не имеет очевидного соответствия в Ozon —
предложи ближайший вариант и отметь: [АДАПТИРОВАНО: было X → стало Y].
Если данных для поля нет совсем — пиши [УТОЧНИТЬ У ПРОДАВЦА].
```
