Для каждого из топ-5 конкурентов из Промпта 3:

1. Открой карточку товара на ozon.ru
2. Прокрути всю галерею фото — убедись что все фото загрузились
3. Сохрани страницу как PDF: Ctrl+P → "Сохранить как PDF"
   Важно: выбери "Фоновая графика" если есть такая опция — чтобы фото сохранились
   Имя файла: card_[название бренда]_[артикул].pdf

  import sys, glob

  # Словарь: номер конкурента → (рейтинг, кол-во отзывов)
