# Промт: Анализ рынка категории (Ozon)

## Назначение
Анализ рынка на основе 28-дневного отчёта продаж из Ozon ЛК (раздел «Аналитика → Что продают конкуренты»).

## Входные данные
- Экспорт из Ozon ЛК: Аналитика → Что продают конкуренты → период 28 дней → CSV/XLSX
- Сохранить в: `data/market/{категория}_{дата}.csv`

## Промт

---

Ты — аналитик маркетплейсов. На основе предоставленных данных за 28 дней проведи анализ рынка.

Категория: {КАТЕГОРИЯ_ТОВАРА}

Задача — предоставить структурированный отчёт:

1. 🎯 ФИЛЬТРАЦИЯ ТОВАРОВ ПО КАТЕГОРИИ
   - Таблица: товары В категории (бренд, примеры, оборот за 28 дней)
   - Таблица: товары ИСКЛЮЧЁННЫЕ (тип, примеры, оборот)
   - Итоговый оборот по категории

2. 💰 ЁМКОСТЬ РЫНКА
   - Оборот за 28 дней, прогноз на месяц и год
   - Средний чек, количество заказов в месяц, количество SKU
   - Доля в общей категории (таблица по сегментам)

3. 📝 НАПРАВЛЕННОСТЬ В КАРТОЧКАХ
   - Таблица: ключевые тренды в позиционировании (направление, % товаров, примеры формулировок)
   - Таблица: структура описаний топ-50 товаров (элемент, наличие %, пример)
   - Таблица: визуальные акценты в карточках (элемент, % товаров)

4. 🏆 ТОП-10 БРЕНДОВ В КАТЕГОРИИ
   - Таблица: место, бренд, оборот/мес, доля рынка, средняя цена, рейтинг

5. 💡 РЕКОМЕНДАЦИИ
   - Таблица: что обязательно указать в карточке (элемент, приоритет, почему)
   - Оптимальная структура описания
   - Ценовые рекомендации по сегментам

6. 📋 ИТОГОВЫЕ ВЫВОДЫ
   - Таблица: ключевые параметры рынка
   - Рекомендация по позиционированию

🔑 ПРИНЦИПЫ:
- Все цифры брать только из предоставленных данных
- Прогнозы помечать как «прогноз»
- Тренды обосновывать конкретными примерами из данных

---

## Расписание
Запускать: еженедельно (данные за 28 дней обновляются еженедельно)

## Запуск
Скажи Claude Code: "Анализ рынка {категория}" и предоставь экспортированный файл из Ozon ЛК
