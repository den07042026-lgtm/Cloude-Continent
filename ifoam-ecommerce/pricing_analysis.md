# Промт: Анализ цен и ценовые рекомендации (Ozon)

## Назначение
Расчёт рекомендуемых цен на основе данных о конкурентах, себестоимости, комиссиях и логистике.

## Входные данные
- `data/pricing/ozon_products_{дата}.xlsx` — выгрузка из Ozon API (скрипт ozon_products.py)
- `data/market/{категория}_{дата}.csv` — данные конкурентов из Ozon ЛК
- Файл с себестоимостью товаров (если есть)

## Промт

---

Ты — аналитик по ценообразованию для маркетплейса Ozon.

На основе предоставленных данных проведи анализ цен и дай рекомендации.

Входные данные:
- Таблица товаров с себестоимостью, текущими ценами, комиссиями и логистикой
- Данные цен конкурентов по категориям

Задача — предоставить:

### 1. Таблица конкурентов по категориям
Только жидкостные аналоги (без аэрозолей и наборов):
| Категория | Товар-конкурент | Цена, ₽ | Объём | Цена за 1 л, ₽ | Продажи, шт |

### 2. Итоговая таблица рекомендуемых цен
Для каждого артикула:
| Артикул | Наименование | Себест., ₽ | Рек. цена, ₽ | Комиссия МП, % | Комиссия МП, ₽ | Логистика FBO, ₽ | Реклама (5%), ₽ | Прибыль с ед., ₽ | Маржа, % | Конкуренты (цены) | Комментарий |

### 3. Принципы расчёта
- Минимальная маржа: 10% с рекламой, 14% без рекламы
- Реклама: закладывать 5% от цены, если маржа позволяет
- Сравнение: цена за единицу объёма (₽/л) vs конкуренты
- Целевое позиционирование: ниже LAVR/DETAIL, выше noname

### 4. Формат комментариев
Для каждого товара объяснить:
- Почему именно такая цена
- Позиция относительно конкурентов (цена за литр)
- Рекомендация по рекламе

🔑 ПРИНЦИПЫ:
- Все цифры брать только из предоставленных данных
- Маржу считать: (Цена - Себест. - Комиссия - Логистика - Реклама) / Цена × 100%
- Не рекомендовать цены ниже порога окупаемости
- Отдельно указывать товары с критически низкой маржой (<8%)
