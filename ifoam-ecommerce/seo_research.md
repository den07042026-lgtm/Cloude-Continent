# Промт: SEO-исследование товара (Ozon)

## Назначение
Составить список ключевых слов и хештегов для карточки товара на Ozon.

## Входные данные
- Название товара и его характеристики
- Данные из аналитики Ozon ЛК (поисковые запросы, если есть)
- Отзывы конкурентов (50-100 шт.) — из файла или вставленные текстом

## Промт

---

Задача: составить список из 30-50 ключевых слов и хештегов для маркетплейсов (Ozon).

Товар: {НАЗВАНИЕ_ТОВАРА}

Учти:
1. Тип товара (тип продукта, форм-фактор)
2. Объём/размер/количество
3. Назначение (для чего, для кого, условия применения)
4. Особенности состава и формулы
5. Популярные поисковые запросы в категории
6. Синонимы и вариации написания

Формат вывода:
- Высокочастотные запросы (10 шт.)
- Среднечастотные запросы (15 шт.)
- Низкочастотные/длинные запросы (15 шт.)
- Хештеги для соцсетей (10 шт.)

Также по отзывам конкурентов (50-100 отзывов):
- Таблица: боли покупателей и решения
- Таблица: конкуренты и их товары с ценами
