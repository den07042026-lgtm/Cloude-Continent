# Промт: ТЗ на анализ и запуск карточки товара (Ozon)

## Назначение
Формирует техническое задание на полный цикл предстартового анализа нового товара —
от извлечения данных этикетки до готового листинга на Ozon.

Структура документа строго соответствует `config/Шаблон_ТЗ.docx`:
5 разделов + 2 таблицы (вводные данные, этапы выполнения).

## Входные данные (предоставить перед запуском)
- Артикул товара (обязательно)
- Бренд (напр. `ifoamHOME`, `ifoamAUTO`)
- Название для папки/slug (напр. `КондиционерBalmy_РайскиеЦветы`)
- Читаемое название товара (напр. `balmy Кондиционер для белья «Райские цветы»`)
- Объём / масса (напр. `1 л`, `5 кг`)
- Дата (опционально — по умолчанию сегодня)

Если карточка товара `01_card_[арт].docx` уже существует — Claude читает её автоматически
и подставляет правильные характеристики в документ.

## Запуск

### Через командную строку (рекомендуется)
```bash
python scripts/gen_tz_product.py \
  --art 770758 \
  --brand ifoamHOME \
  --name "КондиционерBalmy_РайскиеЦветы" \
  --product_name "balmy Кондиционер для белья «Райские цветы»" \
  --volume "1 л" \
  --date "31.03.2026"
```

### Через Claude Code (если нужно добавить характеристики из карточки)
> "Создай ТЗ для артикула 770758, бренд ifoamHOME, продукт balmy Кондиционер «Райские цветы» 1 л"

Claude читает `01_card_[арт].docx`, дополняет ТЗ характеристиками товара
и вызывает `scripts/gen_tz_product.py`.

```

Артикул:        [арт]

Объём/масса:    [объём]
Дата:           [ДД.ММ.ГГГГ или "сегодня"]
============

АВТОДОПОЛНЕНИЕ (выполни до генерации):

1. Найди файл карточки товара:
   C:/Users/1/Downloads/[бренд]_[slug]_[арт]/01_card_[арт].docx
   Если существует — прочитай и извлеки:
     - Назначение одной фразой
     - Ключевые характеристики (pH, концентрация, дозировка, стирки/применения)
     - Состав (краткий)
   Передай как extra_chars в build_product_tz()

2. Определи папку продукта:
   C:/Users/1/Downloads/[бренд]_[slug]_[арт]/

ВЫПОЛНИ:

  import sys

  from gen_tz_product import build_product_tz

  build_product_tz(
      folder=folder,
      art="[арт]",
      brand="[бренд]",
      name="[slug]",
      product_name="[полное название]",
      volume="[объём]",
      date_str="[дата]",
      extra_chars={
          "Назначение":      "[из карточки]",
          "pH":              "[из карточки]",
          "Дозировка":       "[из карточки]",
          "Состав (кратко)": "[из карточки]",
      }
  )
```

## Выходной файл

### Структура документа (по Шаблон_ТЗ.docx)

| Раздел | Содержимое |
|--------|-----------|
| 1. Стороны | Заказчик / Исполнитель / Кто использует |
| 2. Цель задачи | Полный предстартовый анализ + комплект документов |
| 3. Вводные данные | Таблица: этикетка, история продаж, конкуренты, PDF-файлы, справочники |
| 4. Этапы выполнения | Таблица 10 шагов (1 → 1a → 1б → 2 → 3 → 4 → 5 → 6 → 7 → 8) |
| 5. Условия выполнения | 7 критериев приёмки результата |
| + Структура папки | Дерево файлов продукта |
| + Характеристики | Ключевые данные из этикетки (если переданы) |
