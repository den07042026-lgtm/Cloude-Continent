## ГАЙД ДЛЯ ОПЕРАТОРА: заполнение характеристик в Ozon Seller

### Где заполнять
seller.ozon.ru → Товары → [артикул] → Редактировать → Характеристики

---

### Обязательные характеристики для автозапчастей

| Характеристика Ozon | Откуда брать | Пример |
|---------------------|-------------|--------|
| **Название** | Промпт 8 (вариант А) | Фильтр масляный Bosch W7000 Toyota Camry 2.5 |
| **Бренд** | Карточка запчасти → "Бренд детали" | Bosch |
| **Применимость** | Карточка → "Применяемость" | Toyota Camry 2018-2023; Lexus ES 250 2019-2022 |
| **OEM номер** | Карточка → "OEM номер" | 0451103369 |
| **Кросс-номера** | Карточка → "Кросс-номера" | W7000; OP641; PH4967 |
| **Тип запчасти** | Карточка → "Категория запчасти" | Фильтр масляный |
| **Страна производства** | Карточка → "Страна производства" | Германия |
| **Состояние** | Карточка → "Состояние" | Новый |
| **Оригинал/аналог** | Карточка → "Оригинал или аналог" | Aftermarket (аналог) |
| **Материал** | Карточка → "Материал" | Металл, бумага |

---

### Поле "Применимость" — критически важно

Ozon индексирует по этому полю. Заполнять максимально полно:
- Марка (Toyota, Volkswagen, BMW...)
- Модель (Camry, Golf, 3 Series...)
- Год (2018-2023 или конкретный год)
- Объём двигателя (2.0, 2.5, 3.0...)
- Тип двигателя (бензин, дизель, гибрид)

**Формат:** `Марка Модель [год] [объём] [тип]; Марка2 Модель2 ...`

**Пример для масляного фильтра:**
```
Toyota Camry 2.5 2018-2023 бензин; Toyota Camry 3.5 2018-2023 бензин;
Lexus ES 250 2019-2022 бензин; Lexus ES 350 2019-2022 бензин
```

---

### Поле "OEM номер" — главный SEO-фактор

Покупатели ищут по OEM. Вводить:
- Основной OEM номер производителя автомобиля
- Кросс-номера через точку с запятой (все аналоги)

**Пример:**
```
0451103369; W7000; OP641; PH4967; E115H D52; OC69
```

---

### Категория в дереве Ozon

Автозапчасти → [Марка авто] → [Тип запчасти]

Примеры:
- Автозапчасти → Фильтры → Масляные фильтры
- Автозапчасти → Тормозная система → Тормозные колодки
- Автозапчасти → Подвеска и рулевое → Амортизаторы

**Важно:** правильная категория = правильные характеристики + нужные покупатели.
Если не уверен — смотри в какой категории топ-продавцы по этой запчасти.
