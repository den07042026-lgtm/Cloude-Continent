# ПРОМПТ 6 — Визуальный анализ карточек конкурентов
# Запускать в: любой Claude с Vision (desktop / web)
# Вход: СПИСОК КОНКУРЕНТОВ + PDF карточек топ-5 конкурентов
# Выход: 06_visual_[артикул].docx — визуальные наблюдения, референсы, рейтинги

---

## ЧТО СКАЧАТЬ ОПЕРАТОРУ (до запуска промпта)

1. Взять топ-5 конкурентов из СПИСКА КОНКУРЕНТОВ (Промпт 3)
2. Открыть карточку каждого на Ozon
3. Прокрутить всю страницу карточки (фото, описание, характеристики)
4. Chrome → Ctrl+P → "Сохранить как PDF"
5. Также скопировать рейтинг и количество отзывов из каждой карточки
6. Прикрепить PDF(ы) к сообщению
