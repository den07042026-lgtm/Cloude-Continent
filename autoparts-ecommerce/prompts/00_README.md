## СХЕМА РАБОТЫ

```
Фото коробки / каталожный лист / описание детали
      │ (оператор прикрепляет файл или вставляет текст)
      ▼
[ПРОМПТ 1] Извлечение данных о запчасти
      │ → 01_part_[артикул].docx
      │ → КАРТОЧКА ЗАПЧАСТИ (текст — скопировать)
      │
      ▼
[ПРОМПТ 1a] ТЗ на анализ товара  ← gen_tz_product.py
      │ → 01a_tz_[артикул].docx  (стороны, цель, вводные, этапы, условия)
      │
      ▼
[ПРОМПТ 1b] Инструкция по заполнению характеристик в Ozon Seller
      │ → ГАЙД ДЛЯ ОПЕРАТОРА (вручную)
      │
      ▼ (оператор скачивает XLSX из DataLens или combined_marketplaces)
[ПРОМПТ 2] История продаж — только данные
      │ → 02_sales_history_[артикул].docx (данные, тренды, аномалии)
      │
      ▼ (оператор скачивает XLSX с ozon-bestsellers по категории)
[ПРОМПТ 3] Анализ конкурентов — данные и цены
      │ → 03_competitors_[артикул].xlsx
      │ → СПИСОК КОНКУРЕНТОВ (текст — скопировать)
      │
      ▼ + ИСТОРИЯ ПРОДАЖ + СПИСОК КОНКУРЕНТОВ
[ПРОМПТ 3b] Стратегия ценообразования
      │ → 03b_pricing_[артикул].xlsx  (сценарии / конкуренты / история)
      │ → 03b_pricing_strategy_[артикул].docx  (сегментация + выводы)
      │ → ПРОФИЛЬ ПРОДАЖ + ЦЕНОВАЯ СТРАТЕГИЯ (текст — скопировать)
      │
      ▼ + СПИСОК КОНКУРЕНТОВ
[ПРОМПТ 4] Анализ отзывов конкурентов
      │ → 04_pains_[артикул].docx
      │ → БОЛИ И ЖЕЛАНИЯ (текст — скопировать)
      │
      ▼ (оператор сохраняет страницу all-queries как PDF)
[ПРОМПТ 5] Поисковые запросы и SEO
      │ → 05_seo_[артикул].xlsx
      │ → SEO ДАННЫЕ (текст — скопировать)
      │
      ▼ (оператор сохраняет карточки конкурентов как PDF)
[ПРОМПТ 6] Визуальный анализ карточек конкурентов
      │ → 06_visual_[артикул].docx
      │ → ВИЗУАЛЬНЫЕ НАБЛЮДЕНИЯ + РЕФЕРЕНСЫ (текст — скопировать)
      │
      ▼ (все данные собраны)
[ПРОМПТ 7] ТЗ для дизайнера
      │ → 07_designer_tz_[артикул].docx
      │ → 07_designer_tz_[артикул].pptx  (презентация по слайдам)
      │
      ▼
[ПРОМПТ 8] Полный листинг товара
            → 08_listing_[артикул].docx
```

## ЧТО СКАЧИВАЕТ / СОХРАНЯЕТ ОПЕРАТОР

| Шаг | Действие оператора | Тип файла |
|-----|--------------------|-----------|
| Перед Промптом 1 | Фото коробки / PDF каталожного листа / описание с сайта поставщика | PNG, PDF или текст |
| Перед Промптом 2 | Скачать отчёт из DataLens (опционально) | XLSX |
| Перед Промптом 3 | Скачать XLSX с ozon-bestsellers (28 дней) по нужной категории | XLSX |
| Перед Промптом 4 | Сохранить страницы отзывов топ-5 конкурентов | PDF × 5 |
| Перед Промптом 5 | Сохранить страницу all-queries | PDF |
| Перед Промптом 6 | Сохранить карточки топ-5 конкурентов | PDF × 5 |

**Как сохранить страницу как PDF:** Chrome → `Ctrl+P` → "Сохранить как PDF"
**Как скачать XLSX с bestsellers:** seller.ozon.ru → Аналитика → Конкуренты → Ozon Bestsellers → Экспортировать

## ОСОБЕННОСТИ АВТОЗАПЧАСТЕЙ vs БЫТОВАЯ ХИМИЯ

| Аспект | Химия (IFOAM) | Автозапчасти |
|--------|--------------|-------------|
| Идентификация | Артикул + объём | OEM номер + применяемость + бренд |
| Главный фильтр конкурентов | Объём (±50%) | OEM-совместимость + категория + применяемость |
| SEO | Назначение + состав | Марка/модель/год + тип детали + OEM |
| Карточка товара | pH, состав, меры предосторожности | OEM, совместимость, материал, размеры |
| Сезонность | Слабая | Выраженная (весна/осень — пики) |
| Конкуренция | По цене и бренду | По OEM-совместимости, бренду оригинала vs аналог |

## ПРАВИЛА

1. Каждый промпт начинается с `=== ВХОД ===` — вставь данные из предыдущего шага
2. После каждого промпта Claude автоматически сохраняет документ в папку продукта
3. Промпты 1, 1a, 2, 3b, 4, 7, 8 — запускать здесь (Claude Code)
4. Промпты 3, 5, 6 — требуют файлов которые оператор скачивает/сохраняет вручную
5. Промпты 1b — инструкции для оператора (что вручную вставить в Ozon Seller)
6. Путь к папке продукта определяется в Промпте 1 и используется во всех остальных
