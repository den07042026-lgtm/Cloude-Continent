## ЧТО СКАЧАТЬ ОПЕРАТОРУ (до запуска промпта)

1. seller.ozon.ru → Аналитика → Поисковые запросы
2. Введи в поиск: [OEM номер] или [марка + модель + тип детали]
3. Открой раздел "Все запросы" — прокрути до конца
4. Chrome → Ctrl+P → "Сохранить как PDF"
5. Также полезно: введи название детали на русском и сохрани тот PDF тоже
6. Прикрепить PDF(ы) к сообщению

## ПРОМПТ

```
=== ВХОД: КАРТОЧКА ЗАПЧАСТИ ===
[из Промпта 1]

=== ВХОД: PDF ПОИСКОВЫХ ЗАПРОСОВ ===
[прикрепи PDF-файл(ы) с поисковыми запросами из Ozon]
=====================================

=== ПАПКА ПРОДУКТА ===
[путь из Промпта 1]
=====================

Извлеки поисковые запросы из PDF и проанализируй их.

ЧАСТЬ А — ИЗВЛЕЧЕНИЕ ЗАПРОСОВ:

Из каждого PDF извлеки все строки в формате:
  запрос | частота | [другие метрики если есть]

Объедини все запросы в единый список, удали дубликаты.

ЧАСТЬ Б — КЛАССИФИКАЦИЯ:

Сгруппируй запросы по типам — специфично для автозапчастей:

  ГРУППА 1 — ПО OEM/КРОСС-НОМЕРУ:
  (самые горячие — покупатель точно знает что ищет)
  - [OEM номер] → [частота]
  - [кросс-номер 1] → [частота]
  ...

  ГРУППА 2 — ПО МАРКЕ + МОДЕЛИ + ТИПУ ДЕТАЛИ:
  (основной трафик, высокая конверсия)
  - [марка] [модель] [тип детали] → [частота]
  - [марка] [год] [тип детали] → [частота]
  ...

  ГРУППА 3 — ПО ТИПУ ДЕТАЛИ + БРЕНДУ:
  (покупатель знает бренд)
  - [бренд] [тип детали] → [частота]
  ...

  ГРУППА 4 — ПО ТИПУ ДЕТАЛИ (широкие запросы):
  (высокий трафик, низкая конверсия, используем в описании)
  - [тип детали] → [частота]
  ...

  ГРУППА 5 — ПРОЧИЕ / НИШЕВЫЕ:
  - ...

ЧАСТЬ В — ПРИОРИТИЗАЦИЯ:

=== SEO ДАННЫЕ ===
Деталь: [название] | OEM: [OEM] | Применяемость: [марка/модель]

ТОП-5 ГЛАВНЫХ ЗАПРОСОВ (вставить в название):
1. [запрос] — [частота]
2. [запрос] — [частота]
3. [запрос] — [частота]
4. [запрос] — [частота]
5. [запрос] — [частота]

ТОП-15 ЗАПРОСОВ ДЛЯ ОПИСАНИЯ:
[пронумерованный список с частотами]

РЕКОМЕНДОВАННОЕ НАЗВАНИЕ (до 200 символов):
[составь название, включив: бренд + тип детали + OEM + марка + модель + год]

СТРУКТУРА SEO ДЛЯ НАЗВАНИЯ:
  [Тип детали] [Бренд] [OEM/кросс-номер] [Марка] [Модель] [Год/объём]
  Пример: Фильтр масляный Bosch 0451103369 Toyota Camry 2.5 2018-2023

ХЭШТЕГИ (8–12 штук):
  #[тип_детали] #[бренд] #[марка_авто] #[модель_авто] #[oem_номер] ...

КЛЮЧЕВЫЕ ИНСАЙТЫ:
- OEM-запросы составляют __% трафика — [вывод]
- Запросы по применяемости: [какие марки/модели чаще всего ищут]
- Брендовые запросы: [популярен ли наш бренд в поиске?]
- Запросы конкурентов: [кого ищут вместо нас?]
```
